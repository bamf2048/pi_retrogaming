# Pi Zero Speed Tips

- set to 720p:

```
hdmi_group=1
hdmi_mode=4
```

- make sure your usb hub isn't drawing too much power (compare speeds by unplugging/plugging in)
- make sure you have a good power supply - do you get the lighting bolt in the top right corner?
